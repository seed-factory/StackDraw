# StackDraw - Инструмент для изометрических диаграмм

[English](../README.md) | [Русский](README.ru.md) | [Deutsch](README.de.md)

StackDraw — это мощное прогрессивное веб-приложение (PWA) с открытым исходным кодом для создания красивых изометрических диаграмм. Построенное на React и TypeScript, оно полностью работает в вашем браузере с поддержкой офлайн-режима.

> Форк [FossFLOW](https://github.com/stan-smith/FossFLOW) от @stan-smith, основанный на библиотеке [Isoflow](https://github.com/markmanx/isoflow) от @markmanx

## Демо

**Попробуйте сейчас:** [https://seed-factory.github.io/StackDraw](https://seed-factory.github.io/StackDraw)

> **Рекомендуемый браузер:** Для лучшего опыта мы рекомендуем использовать **Google Chrome** или другие браузеры на основе Chromium (Edge, Brave, Opera). Некоторые функции, такие как экспорт изображений, могут иметь проблемы совместимости в Firefox.

## Возможности

- Создание изометрических диаграмм с интерфейсом drag-and-drop
- Соединители с настраиваемыми метками и стилями
- Поддержка нескольких наборов иконок (AWS, GCP, Azure, Kubernetes)
- Экспорт в PNG, SVG, JSON
- PWA с поддержкой офлайн-режима
- Многоязычный интерфейс (12+ языков)
- Хранение в сессии и на сервере
- История версий с Git-бэкапом

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/seed-factory/StackDraw
cd StackDraw

# Установить зависимости
npm install

# Собрать библиотеку (требуется при первом запуске)
npm run build:lib

# Запустить сервер разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере.

## Docker

```bash
# Использование Docker Compose (рекомендуется)
docker compose up

# Или запустите напрямую с постоянным хранилищем
docker run -p 80:80 -v $(pwd)/diagrams:/data/diagrams stackdraw:latest

# Отключить серверное хранилище
docker run -p 80:80 -e ENABLE_SERVER_STORAGE=false stackdraw:latest
```

## Структура проекта

Монорепозиторий с тремя пакетами:

- `packages/fossflow-lib` - Библиотека React-компонентов для рисования диаграмм
- `packages/fossflow-app` - Прогрессивное веб-приложение
- `packages/fossflow-backend` - Node.js бэкенд для серверного хранилища

### Команды

```bash
# Разработка
npm run dev          # Запустить сервер разработки приложения
npm run dev:lib      # Режим наблюдения для библиотеки

# Сборка
npm run build        # Собрать библиотеку и приложение
npm run build:lib    # Собрать только библиотеку
npm run build:app    # Собрать только приложение

# Тестирование
npm test             # Запустить модульные тесты
npm run lint         # Линтинг
```

## Использование

### Создание диаграмм

1. **Добавить элементы**: Нажмите кнопку "+", перетащите компоненты из библиотеки на холст
2. **Соединить элементы**: Выберите инструмент Соединитель (C), щелкните по узлам для соединения
3. **Сохранить**: Быстрое сохранение в сессию или экспорт как JSON

### Варианты хранения

- **Хранилище сессии**: Временное, очищается при закрытии браузера
- **Экспорт/Импорт**: Постоянные JSON-файлы
- **Серверное хранилище**: Постоянное с историей версий (Docker)
